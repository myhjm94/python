# input函数：你的第一个“麦克风”🎤

## 一、input函数是什么？

想象一下，input就像是程序世界的**麦克风**或者**提问器**。当我们想让电脑从键盘接收我们输入的信息时，就用它来“听我们说话”或“问我们问题”。

### 最简单的用法：
```python
input("请输入你的名字：")
```
运行后，屏幕上会显示：
> **请输入你的名字：**

然后电脑会等待你输入，比如你输入“小明”后按回车，电脑就知道了你的名字！

---

## 二、基本用法（很好玩的哦！）

### 1. 最简单的提问
```python
input("你今年几岁了？")
```
运行后：
> **你今年几岁了？**  
（然后你可以输入：10，然后按回车）

### 2. 把回答存起来
```python
name = input("你叫什么名字？")
print("你好，", name, "！")
```
运行后：
> **你叫什么名字？**（你输入：小红）  
**你好， 小红 ！**

**重点📌：用`=`可以把输入的内容保存到变量里，就像给东西贴标签一样！**

### 3. 不显示问题直接问
```python
age = input()  # 直接等待输入
print("你输入的是：", age)
```
运行后：
> （光标闪烁，等待你输入）

## 三、超级有用的功能✨

### 1. 问多个问题
```python
name = input("姓名：")
age = input("年龄：")
school = input("学校：")

print("===学生信息===")
print("姓名：", name)
print("年龄：", age, "岁")
print("学校：", school)
```
运行后：
> **姓名：**（输入：小明）  
**年龄：**（输入：10）  
**学校：**（输入：阳光小学）  
**===学生信息===**  
**姓名： 小明**  
**年龄： 10 岁**  
**学校： 阳光小学**

### 2. 做个小计算器
```python
num1 = input("请输入第一个数字：")
num2 = input("请输入第二个数字：")
print("你输入的是：", num1, "和", num2)
```
**注意⚠️：input收到的都是文字（字符串），不能直接计算！**

## 四、重要！重要！重要！🔴

### 重点1：input得到的都是文字！
```python
age = input("你几岁了？")
print(type(age))  # 显示：<class 'str'>（意思是：文字类型）
```
**重点📌：即使你输入数字10，电脑也把它当作文字"10"，不是数字10！**

### 重点2：想要数字怎么办？
需要把文字变成数字：
```python
age_str = input("你几岁了？")  # 得到文字
age_num = int(age_str)         # 把文字变成数字
print("明年你就", age_num + 1, "岁了！")
```
或者更简单：
```python
age = int(input("你几岁了？"))  # 一句话搞定！
print("明年你就", age + 1, "岁了！")
```

### 重点3：小数怎么办？
```python
height = float(input("你的身高是多少米？"))
print("你的身高是", height, "米")
```

## 五、常见错误要注意！🚨

### 错误1：忘记把文字变数字就计算
```python
age = input("年龄：")
print(age + 1)  # ❌ 错误！文字不能加数字
```
改正：
```python
age = int(input("年龄："))
print(age + 1)  # ✅ 正确！
```

### 错误2：输入的不是数字却要变成数字
```python
name = int(input("名字："))  # ❌ 如果输入"小明"，会出错！
```
改正：
```python
name = input("名字：")  # ✅ 名字就用文字
```

### 错误3：拼写错误
```python
inpt("请输入")  # ❌ 少了u！
input("请输入")  # ✅ 正确
```

## 七、记住这些就够用了！💡

1. input是 **"麦克风"**，用来接收键盘输入
2. input得到的是**文字**，不是数字
3. 要用**int()**或**float()** 把文字变成数字才能计算
4. 用**变量**保存输入，方便后面使用
5. input的**拼写要正确**，一个字母都不能少

## input和print是好伙伴👫

记住：
- **print**是"小喇叭"📢：说给用户听
- **input**是"麦克风"🎤：听用户说

它们经常一起使用：
```python
# 先问问题
name = input("你叫什么名字？")

# 再回答
print("你好，", name, "！欢迎来到Python世界！")
```
