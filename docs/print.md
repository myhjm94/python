# print函数：你的第一个“小喇叭”📢

## 一、print函数是什么？

想象一下，print就像是程序世界的**小喇叭**或者**写字板**。当我们想让电脑在屏幕上显示一些文字或信息时，就用它来“喊出来”或“写出来”。

### 最简单的用法：
```python
print("你好，世界！")
```
运行后，屏幕上会显示：
> **你好，世界！**

---

## 二、基本用法（很简单的哦！）

### 1. 打印文字（字符串）
```python
print("我爱学Python！")
print('用单引号也可以哦！')
```
运行后，屏幕上会显示：
> **我爱学Python！**<br>
  **用单引号也可以哦！**<br>

**重点📌：文字要用引号（双引号或单引号）包起来**

### 2. 打印数字
```python
print(100)
print(3.14)
```
运行后，屏幕上会显示：
> **100**<br>
  **3.14**<br>

**重点📌：数字不用加引号**

### 3. 打印多句话
```python
print("小明", "小红", "小刚")
```
会显示：
> **小明 小红 小刚**

---

## 三、超级有用的功能✨

### 1. 连接文字和数字（用逗号）
```python
print("我的年龄是：", 10, "岁")
```
显示：
> **我的年龄是： 10 岁**

**注意⚠️：逗号会让它们之间有一个空格**

### 2. 去掉空格（sep参数）
```python
print("2024", "03", "15", sep="-")
```
显示：
> **2024-03-15**  
 （而不是 2024-03-15）

### 3. 换行和不换行
默认每次print都会换行：
```python
print("第一行")
print("第二行")
```
显示：
> **第一行**<br>
  **第二行**<br>

如果不想换行：
```python
print("Hello ", end="")
print("World!")
```
显示：
> **Hello World!**

---

## 四、常见错误要注意！🚨

### 错误1：忘记引号
```python
print(你好)   # ❌ 错误！
print("你好") # ✅ 正确
```

### 错误2：中英文标点混用
```python
print(“中文引号”)  # ❌ 错误！
print("英文引号")  # ✅ 正确
```
**重点📌：编程中一定要用英文标点！**

### 错误3：拼写错误
```python
prin("你好")  # ❌ 少了一个t！
print("你好")  # ✅ 正确
```

## 六、记住这些就够用了！💡

1. **打印文字要加引号**（双引号或单引号都可以）
2. **打印数字不用引号**
3. **用逗号分隔多个内容**，它们会一起显示
4. **所有标点要用英文的**
5. **print的拼写要正确**，一个字母都不能少
