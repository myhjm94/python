# 变量：你的第一个“魔法盒子”🎁

## 一、变量是什么？

想象一下，变量就像是程序世界的**魔法盒子**或者**标签贴纸**。它可以**存放东西**（数字、文字、列表等），还可以**贴上标签**（变量名），方便我们找到和使用它！

### 最简单的例子：
```python
name = "小明"  # 创建一个叫name的盒子，里面放了"小明"
age = 10       # 创建一个叫age的盒子，里面放了10
```
这就好比：
- 📦 **name盒子** → 装着"小明"
- 📦 **age盒子** → 装着10

---

## 二、基本用法（就像贴标签一样简单！）

### 1. 创建变量（贴标签）
```python
fruit = "苹果"    # 水果盒子贴着"苹果"标签
number = 100      # 数字盒子贴着100标签
price = 3.5       # 价格盒子贴着3.5标签
```

### 2. 使用变量（看盒子里有什么）
```python
fruit = "香蕉"
print("我喜欢吃", fruit)  # 看看fruit盒子里是什么
```
显示：
> **我喜欢吃 香蕉**

### 3. 修改变量（换盒子里的东西）
```python
fruit = "苹果"
print("现在的水果是：", fruit)

fruit = "橘子"  # 换内容啦！
print("现在的水果是：", fruit)
```
显示：
> **现在的水果是： 苹果**  
> **现在的水果是： 橘子**

**重点📌：变量就像可擦写的白板，可以随时修改内容！**

## 三、变量的命名规则（重要！）📝

### 规则1：名字要有意义
```python
# ✅ 好名字，一看就知道是什么
student_name = "小红"
student_age = 11
math_score = 95

# ❌ 坏名字，看不懂是什么
a = "小红"
b = 11
c = 95
```

### 规则2：只能用字母、数字、下划线
```python
# ✅ 正确的
my_name = "小明"
name1 = "小红"
your_name = "小刚"

# ❌ 错误的
my-name = "小明"  # 不能用横线
1name = "小红"    # 不能以数字开头
my name = "小刚"  # 不能有空格
```

### 规则3：大小写敏感
```python
Name = "小明"  # 大写的N
name = "小红"  # 小写的n

print(Name)  # 显示：小明
print(name)  # 显示：小红
```
**重点📌：Name和name是两个不同的盒子！**

## 四、不同类型的盒子📦

### 1. 文字盒子（字符串）
```python
name = "小明"           # 普通文字
hobby = '打篮球'        # 单引号也可以
sentence = "我喜欢Python"  # 一句话
```

### 2. 数字盒子（整数和浮点数）
```python
age = 10          # 整数盒子
height = 1.45     # 小数盒子（浮点数）
score = 100       # 整数盒子
pi = 3.14159      # 小数盒子
```

### 3. 布尔盒子（对/错）
```python
is_rain = True    # 真的（下雨了）
is_sunny = False  # 假的（没出太阳）
```

## 五、变量的魔法操作✨

### 1. 盒子之间可以传递东西
```python
a = 5      # a盒子放5
b = a      # b盒子也放5（复制一份）
print(a, b)  # 显示：5 5
```

### 2. 盒子内容可以交换
```python
x = "苹果"
y = "香蕉"
print("交换前：", x, y)

# 魔法交换！
x, y = y, x
print("交换后：", x, y)
```
显示：
> **交换前： 苹果 香蕉**  
> **交换后： 香蕉 苹果**

### 3. 盒子可以一起创建
```python
# 一次创建多个盒子
name, age, score = "小明", 10, 95
print(name, age, score)
```

## 六、常见错误要注意！🚨

### 错误1：盒子没放东西就用
```python
print(score)  # ❌ 错误！score盒子还没创建
```
改正：
```python
score = 100    # ✅ 先创建盒子
print(score)   # ✅ 再使用盒子
```

### 错误2：错别字（大小写错误）
```python
Name = "小明"
print(name)  # ❌ 错啦！应该是Name不是name
```
改正：
```python
Name = "小明"
print(Name)  # ✅ 正确！
```

### 错误3：用Python的关键字当名字
```python
print = "你好"  # ❌ 错误！print是特殊词
if = 10         # ❌ 错误！if是特殊词
```
**重点📌：这些词不能当变量名：** print, input, if, for, while, True, False

## 八、记住这些就够用了！💡

1. **变量是魔法盒子**📦，可以存放各种东西
2. **名字要有意义**，方便自己和别人看懂
3. **先创建盒子**，再使用盒子里的东西
4. **大小写要分清**，Name和name不一样
5. **不能用特殊词**当变量名

## 九、变量的好朋友👫

变量经常和这些朋友一起玩：

```python
# 和print一起玩
name = "小明"
print("你好，", name)  # 显示盒子的内容

# 和input一起玩
age = input("你几岁了？")  # 把输入放进盒子
print("你", age, "岁了")   # 从盒子里拿出来显示

# 和计算一起玩
a = 10
b = 20
c = a + b  # 把计算结果放进新盒子
print("10 + 20 =", c)
```

## 一句话总结💬

**变量就是: 名字 + 值**

就像：
- **名字**：盒子上的标签（student_name）
- **值**：盒子里的东西（"小明"、10、95.5等）

记住这个公式，你就掌握了变量的魔法！🎉
