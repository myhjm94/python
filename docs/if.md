# 分支结构：你的第一个“选择题”🤔

## 一、分支结构是什么？

想象一下，分支结构就像是程序世界的**选择题**或者**分叉路口**。当程序遇到不同情况时，它可以**选择不同的路**继续走下去，就像你做选择题一样！

### 最简单的例子：
如果今天下雨🌧️ → 带伞出门  
如果今天晴天☀️ → 不带伞出门

---

## 二、if语句：最简单的选择题

### 1. 基本格式
```python
if 条件:
    做某件事
```

就像说：**如果**条件成立，**那么**就做这件事。

### 2. 第一个if程序
```python
age = 10

if age >= 10:
    print("你可以玩这个游戏！")
```
显示：
> **你可以玩这个游戏！**

**重点📌：if后面的冒号(:)不能少！下面的代码要缩进（按Tab键或4个空格）**

---

## 三、if-else：二选一的选择题

### 1. 基本格式
```python
if 条件:
    做第一件事
else:
    做第二件事
```

就像说：**如果**条件成立，**那么**做第一件事，**否则**做第二件事。

### 2. 例子：检查成绩
```python
score = 85

if score >= 60:
    print("恭喜！你及格了！")
else:
    print("很遗憾，需要继续努力！")
```
因为85>=60，所以显示：
> **恭喜！你及格了！**

如果`score = 55`，就会显示：
> **很遗憾，需要继续努力！**

---

## 四、if-elif-else：多选一的选择题

### 1. 基本格式
```python
if 条件1:
    做第一件事
elif 条件2:
    做第二件事
elif 条件3:
    做第三件事
else:
    做其他事
```

就像考试评级：90分以上→A，80-89→B，70-79→C，其他→D

### 2. 例子：成绩评级
```python
score = 88

if score >= 90:
    print("优秀！得A")
elif score >= 80:
    print("良好！得B")
elif score >= 70:
    print("中等！得C")
elif score >= 60:
    print("及格！得D")
else:
    print("不及格！要加油！")
```
因为88>=80且<90，所以显示：
> **良好！得B**

## 五、比较运算符：怎么比较？⚖️

| 运算符 | 意思 | 例子 | 结果 |
|:--------:|:------:|:------:|:------:|
| > | 大于 | 10 > 5 | True |
| < | 小于 | 3 < 2 | False |
| >= | 大于等于 | 10 >= 10 | True |
| <= | 小于等于 | 5 <= 3 | False |
| == | 等于 | 5 == 5 | True |
| != | 不等于 | 5 != 3 | True |

**重点📌：比较相等要用`==`（两个等号），不是`=`（一个等号）！**

## 六、逻辑运算符：多个条件一起判断🔗

### 1. and（并且）
```python
age = 12
height = 1.5

if age >= 10 and height >= 1.4:
    print("你可以玩过山车！")
else:
    print("你还不能玩过山车")
```
显示：
> **你可以玩过山车！**

### 2. or（或者）
```python
weather = "阴天"

if weather == "晴天" or weather == "阴天":
    print("适合户外活动")
else:
    print("最好待在家里")
```
显示：
> **适合户外活动**

### 3. not（不）
```python
is_raining = False

if not is_raining:  # 如果不下雨
    print("可以去操场玩")
else:
    print("在教室看书")
```
显示：
> **可以去操场玩**

## 七、常见错误要注意！🚨

### 错误1：忘记冒号
```python
if age > 10  # ❌ 错误！少了冒号
    print("可以")
```
改正：
```python
if age > 10:  # ✅ 正确！有冒号
    print("可以")
```

### 错误2：忘记缩进
```python
if age > 10:
print("可以")  # ❌ 错误！没缩进
```
改正：
```python
if age > 10:
    print("可以")  # ✅ 正确！有缩进
```

### 错误3：用=代替==
```python
if age = 10:  # ❌ 错误！应该用==
    print("10岁")
```
改正：
```python
if age == 10:  # ✅ 正确！用==比较
    print("10岁")
```

## 九、记住这些就够用了！💡

1. **if是选择题**，让程序做决定
2. **冒号不能少**，就像句子结束要句号
3. **缩进很重要**，表示哪些代码属于哪个选择
4. **比较用==**，不是=（一个等号是赋值）
5. **可以组合条件**，用and、or、not

## 一句话总结💬

**分支结构 = 如果...那么...否则...**

就像：
- **如果**今天下雨 → **那么**带伞
- **否则如果**今天很热 → **那么**带水
- **否则** → 什么都不带

记住这个思考方式，你就能让程序做选择题了！🤔
